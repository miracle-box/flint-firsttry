This document is used to test whether the outdated document is working properly.

## Look carefully

We set only `outdated: true` in the default locale, and set only `outdated_translation: "https://github.com/miracle-box/astro-theme-rocket"` in non-default locales.

There will be an outdated prompt in the default locale, and both outdated and outdated translation prompt in non-default locales.
